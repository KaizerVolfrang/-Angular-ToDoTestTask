# ToDoTestTask

Простое ToDo-приложение на Angular, выполненное в рамках тестового задания. Реализовано добавление, удаление и отметка задач как выполненных. Данные сохраняются в LocalStorage. Присутствует валидация ввода (пустые строки, дубликаты). Используются standalone-компоненты, двусторонняя привязка и базовая стилизация.

## Установка и запуск

1. Клонируйте репозиторий:
   `git clone https://github.com/KaizerVolfrang/ToDoTestTask.git`
2. Установите зависимости:
   `npm install`
3. Запустите проект:
   `ng serve`
4. Перейдите в браузере на `http://localhost:4200/`

## Стек технологий

- Angular 20 (standalone компоненты)
- TypeScript
- HTML/CSS
- LocalStorage API

## Примечание

Папка `node_modules` не включена в репозиторий. Для запуска проекта выполните `npm install`.
